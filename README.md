# Frontend Design Challenge – APS Security Dashboard

This project is a frontend implementation of a cybersecurity dashboard developed as part of a technical design challenge. The objective of this task was to recreate product screens based on a design reference and build a fully interactive React application.

The application represents a security scanning platform where users can view scans, monitor progress, and observe simulated scan activity through a clean and modern interface.

---

## Overview

The project focuses on building a responsive and interactive dashboard experience using reusable components and structured frontend architecture. The interface demonstrates how users would interact with a real-world cybersecurity scanning system.

---

## Features

- User signup and login interface
- Dashboard displaying scan overview
- Multiple scan records displayed in a structured table
- Clickable scan rows with navigation to scan details
- Scan details page with progress section
- Simulated live console output
- Dark and Light theme toggle
- Responsive layout
- Component-based React architecture

---

## Tech Stack

- React (Vite)
- React Router DOM
- JavaScript (ES6+)
- CSS (Custom Styling)
